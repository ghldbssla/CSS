미디어 쿼리

미디어 쿼리는 화면 크기에 따른 각각의 속성 값을 지정하여, 여러가지 화면을 구성하는 기술이다

@media (only, all, and) (조건문){ 스타일 적용문(실행문) }

  • @media : 미디어 쿼리가 시작됨을 표시
  • @only : 미디어 쿼리 구문을 해석해라는 명령어(생략 가능)
  • @all : 미디어 쿼리를 해석해야 할 대상을 나타냄(생략 가능)
  • and : 앞과 뒤의 조건을 나타낸다(생략 가능)
  • (조건문) : 해당 조건을 설정
  • (실행문) : 조건에 따른 적용할 속성 설정
